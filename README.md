## üöÄ Overview

**VoidAnime Verse** is a cutting-edge, aesthetically crafted desktop application for anime and manga enthusiasts. Built with performance and beauty in mind, it combines the power of **Go** backend with the flexibility of **Next.js** and **Tauri/Electron** frontend to deliver a seamless, ad-free, and immersive experience.

> [!NOTE]
> This repository is for **App Downloads and Issues** only. The source code for VoidAnime Verse is currently private.

Experience your favorite series like never before with 4K upscaling, torrent streaming, offline capabilities, and a fully customizable interface.

## ‚ú® Key Features

*   **üé¨ Advanced Streaming**:
    *   **Torrent Streaming**: Watch high-quality releases directly via torrents with zero waiting time.
    *   **HLS Proxy**: Seamless reliable streaming with HLS fallback.
    *   **Anime4K Upscaling**: Real-time GPU-accelerated upscaling for crystal clear visuals.

*   **üìö Manga & Novels**:
    *   Integrated manga reader with support for multiple extensions (MangaFire, WeeBCentral, etc.).
    *   Download chapters for offline reading.

*   **üé® Stunning UI/UX**:
    *   Modern, glassmorphic design built with **TailwindCSS** and **Radix UI**.
    *   Fully customizable home page layout.
    *   Wallpaper engine: Set your favorite anime scenes as dynamic backgrounds.

*   **üéµ Music Player**:
    *   Built-in music player to stream and download anime soundtracks.
    *   Offline storage support.

*   **üõ†Ô∏è Power User Tools**:
    *   **Discord RPC**: Show off what you're watching or reading in real-time.
